## Setup

To view this project locally:

1. Clone the repository:
    ```bash
    git clone https://github.com/betty-19/Portfolio.git
    ```

2. Navigate to the project directory:
    ```bash
    cd Portfolio
    ```

3. Open `index.html` in your browser to view the website.
